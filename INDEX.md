## 🚀 Quick Start (3 Steps)

### **Step 1: Open the App**
```bash
# Option 1: Open directly
open index.html

# Option 2: Run local server
python -m http.server 8000
# Then visit: http://localhost:8000
```

### **Step 2: Add Your First Prompt**
- Click the **"+"** button (bottom-right corner)
- Or press **Ctrl+N**
- Fill in the form
- Click **Save Prompt**

### **Step 3: Explore Features**
- Try searching (Ctrl+F)
- Toggle dark mode (🌙 button)
- Switch views (grid/list)
- View statistics (📊 button)
- Explore keyboard shortcuts

## 🆘 Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| Data not saving | Check localStorage is enabled |
| Dark mode not working | Refresh page, check privacy settings |
| Search not finding | Ensure JavaScript is enabled |
| Images not uploading | Verify format (PNG/JPG), check connection |
| UI looks broken | Clear browser cache, refresh |
