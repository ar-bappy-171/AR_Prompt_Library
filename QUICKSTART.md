# 🚀 Quick Start Guide - Prompt Vault

## Getting Started in 3 Steps

### 1️⃣ Open the Application
- Open `index.html` in your web browser
- Or run: `python -m http.server 8000` and visit `http://localhost:8000`

### 2️⃣ Add Your First Prompt
- Click the **"+"** button in the bottom-right corner (Floating Action Button)
- Or use keyboard shortcut: **Ctrl+N**
- Fill in the form:
  - **Title**: Give your prompt a descriptive name
  - **Category**: Choose or create a category
  - **Prompt Text**: Paste your AI prompt here
  - **Tags** (optional): Add comma-separated tags
  - **Rating** (optional): Rate effectiveness (1-5 stars)
  - **Images** (optional): Add example output images
- Click **Save Prompt**

### 3️⃣ Manage Your Prompts
- **Search**: Use the search bar to find prompts by title, text, or tags
- **Filter**: Click categories in the sidebar to filter
- **Copy**: Click copy button or double-click the card to copy prompt
- **Edit**: Click edit button to modify prompt
- **Delete**: Click delete button (with confirmation)

## 📊 Understanding the Dashboard

### Statistics Modal
Click the **📊 Stats** button to see:
- **Total Prompts**: How many you have
- **Total Tags**: Unique tags used
- **With Images**: Prompts with visual examples
- **Categories Used**: How many categories
- **Avg Words**: Average prompt length
- **Avg Rating**: Average effectiveness score
- **Category Breakdown**: Bar chart distribution

### Prompt Card Info
Each card displays:
- **Title**: What the prompt is for
- **Category Badge**: Color-coded category
- **Preview Text**: First 200 characters
- **Stats Bar**: Word count, tokens, complexity, rating
- **Tags**: Organization tags
- **Date**: When created

## 🔒 Data Management

### Automatic Saving
- All changes save to **browser local storage** instantly
- No internet needed
- Data persists across sessions

### Backup Your Data
```
1. Click "Export" button
2. Save the JSON file
3. Store safely as backup
```

### Restore Data
```
1. Click "Import" button
2. Select your JSON backup
3. Confirm to replace current data
```

### Undo Mistakes
```
1. Use Ctrl+Z immediately after mistake
2. Up to 20 undo steps available
3. Works for all operations
```

## ⭐ Rating System Guide

### Star Ratings Meaning
```
⭐ (1 star)        → Poor - Needs significant revision
⭐⭐ (2 stars)      → Fair - Works but needs improvement
⭐⭐⭐ (3 stars)     → Good - Works well for most cases
⭐⭐⭐⭐ (4 stars)    → Very Good - Consistently effective
⭐⭐⭐⭐⭐ (5 stars)   → Excellent - Perfect or near-perfect
```

### How to Rate
1. Click "Edit" on prompt
2. Click stars in rating section
3. Save prompt
4. Rating appears on card

## 🐛 Quick Troubleshooting

### "Data disappeared"
- Check if you're using private/incognito mode
- Local storage may be disabled
- Try exporting data first

### "Search not working"
- Ensure JavaScript is enabled
- Try refreshing the page
- Check browser console (F12)

### "Images not uploading"
- Try PNG or JPG format
- Check internet connection

### "Dark mode not saving"
- Browser may not allow storage
- Try disabling tracking protection
- Use regular (not private) mode
