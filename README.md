# Prompt Vault

AI Prompt Management System

## Overview

Check it out :-- https://ar-bappy-171.github.io/AR_Prompt_Library/

## Quick Start

1. Open `index.html` in your browser
2. Click the **+** button to add a new prompt
3. Fill in the title, category, and prompt text
4. Optionally add tags, images, or notes
5. Click **Save Prompt**

## Usage

- **Add Prompt**: Click **+** or use `Ctrl+N`
- **Search**: Use the search box with real-time filtering
- **Edit**: Click **Edit** button or double-click a prompt

## Files

- `README.md` - Documentation

- **Create & Manage**: Add, edit, and delete prompts with full text support
